# 时间线梳理

## 远古时代

| 时间 | 事件 |
|------|------|
| 天地初分 | 大地裂出缝隙，灵气溢出，各族生物逐渐演化 |
| 万余年后 | 第一批天才修行者找到灵气缝隙，以此为中心建长生殿，周边八城合称九州岛 |
| 灵气亏损期 | 修士长生导致灵气入不敷出，大修士们将九州岛升入天穹，自称仙界 |
| 仙界初期 | 天庭成立，天帝统御万方，开天梯设四劫八兽为升仙试炼 |
| 仙人达阈值 | 仙人数量达一万，天帝提出狩猎模式 |
| 妖魔起源期 | 天界豢养妖魔与魔兽吸收灵气，后被天帝禁止，落入凡间失控 |

## 两千年前

| 时间 | 事件 |
|------|------|
| 两千年前 | 七圣（枪、剑、书、弓、棋、琴、医）绝世天才登仙又返回人间 |
| 此后不久 | 剑圣封闭天梯，断绝升仙之路；将灵孔开大补充凡间灵气 |
| 同期 | 剑圣用结界封印魔族于魔界，九大魔王以自身为中心开九墟 |
| 同期 | 枪圣在地下深处创造灵界，维持灵气循环 |
| 同期 | 书圣进入凡人王朝担任太子老师，守卫凡间秩序 |
| 同期 | 剑圣立下太苍境法令，隐居太苍山后山 |

## 千余年前

| 时间 | 事件 |
|------|------|
| 琴圣封印九墟 | 琴圣在封印九墟时感染魔毒，毒入灵脉而陨落 |
| 此后 | 剑圣启动九世轮回计划，凝聚琴圣灵气、封印记忆、重塑身躯 |
| 千余年间 | 琴圣历经七世轮回，灵脉逐渐纯净化 |
| 弓圣、棋圣、医圣 | 先后陨落（具体时间与原因待定） |

## 近百年

| 时间 | 事件 |
|------|------|
| 约70年前 | 太苍山内乱：后山长老联合掌门以稀世阵法杀剑圣，欲解除太苍境法令，惨败，掌门及一众高手殒命 |
| 60多年前 | 剑圣捏造无记忆分身苏长明（剑仙）拜入太苍山门 |
| 40年前 | 苏长明收玉玲珑（琴圣第八世）为徒 |
| 20年前 | 40岁的玉玲珑到达太苍境，外出历练时击败陈墨，陈墨依据赌约拜玉玲珑为师，成为玉玲珑首徒（后来的隐剑峰大师兄） |

## 10年前（关键转折）

| 事件序列 | 详情 |
|----------|------|
| 1. 前往魔界 | 苏长明带玉玲珑前往魔界解决危机 |
| 2. 镇压心之魔域 | 苏长明以大神通镇压反叛的心之魔域 |
| 3. 发现沈惊蛰 | 发现沈惊蛰身上灵气远强于魔气，判断他不是魔族，将其带出魔界。沈惊蛰母亲死于此役 |
| 4. 镇魔柱危机 | 回山后太苍山隐剑峰镇魔柱开始不稳定 |
| 5. 苏长明牺牲 | 苏长明选择牺牲自己重新加固镇魔柱，肉身消散融入柱中 |
| 6. 玉玲珑受损 | 玉玲珑最后一刻抓住师父的手，一魂一魄被带入镇魔柱，此后只能在隐剑峰一定范围内动用全部法力 |
| 7. 沈惊蛰入门 | 沈惊蛰加入太苍山，拜玉玲珑为师 |
