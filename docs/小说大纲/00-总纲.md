# 《几时休》小说总纲

## 总体定位
- 类型：东方玄幻 / 宗门修真 / 三界博弈
- 核心主线：沈惊蛰从太苍山隐剑峰一名普通弟子起步，经雷池守池、天下会武、西境试炼、心之墟斗法，一步步触及九墟封印、天梯重启与天泉真相，最终重写三界秩序。
- 核心主题：
  - 当资源有限、众生有欲时，秩序应由谁来决定代价。
  - 强者的"正确"是否可以替代众生自己的选择。
  - 真正的休战不是压制一切声音，而是建立可共同承担的规则。
- 核心规则：太苍派从上到下皆为正派，不存在内部反派或暗中作恶的高层，宗门内部矛盾仅限于理念分歧。

## 已吸收的设定
- 境界体系：初照、曲径、幽径、太苍、古道（每升一境增加100年寿命上限），半仙。
- 宗门基础：太苍派为人界正道支柱，五峰结构（问剑峰、听泉峰、落霞峰、青云峰、隐剑峰）。
- 四大门派：太苍派（东）、归墟宗（西）、沧澜阁（南）、霜衡门（北）。
- 关键人物：
  - 隐剑峰：玉玲珑、陈墨、苏念卿、卫朝歌、姜时雨、沈惊蛰
  - 问剑峰：沈渡衡、顾行舟、秦望
  - 其他峰：叶青棠、柳沉渊、裴惊鸿、温如晦
  - 其他门派：韩素（归墟宗）、纪云（沧澜阁）、乔峥（霜衡门）
  - 凡人王朝：萧承安（化名凌天）
  - 魔界：心之墟魔主、厉渊、烛阴、骨卫
  - 已故：苏长明（剑仙）
- 世界底层：天泉供灵、天庭控天、九墟镇魔、剑圣封天、灵界净气。
- 修炼体系：纯剑修，功法三阶段（纳气/炼化/通天），斗法四层（灵识对抗/灵气争夺/术法交锋/法则压制）。
- 关键物品：赤雷剑、引雷佩、破魔之剑。

## 主角成长线
- 前期（第一、二卷）：从普通弟子起步，经守池磨砺、天下会武与西境试炼，学会"守"与"行"——在修行中扎稳根基，在实战中见识天地广阔，开始触及三界秩序的边缘。
- 中期（第三、四、五卷）：亲历超出自身境界的生死危局，目睹师父远超想象的力量，认知被打碎后重建，学会"疑"——开始追问剑圣法令的代价是否公平，正邪观被撬动。
- 后期（第六、七卷）：从被棋局推动的人，成长为主动重写规则的人，学会"立"——不依附任何旧神，也不复制任何旧秩序。
