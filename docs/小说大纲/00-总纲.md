# 《几时休》小说总纲

## 总体定位
- 类型：东方玄幻 / 宗门修真 / 三界博弈
- 核心主线：主角因赤雷剑与剑圣分身结缘，从太苍山一名新秀弟子，一步步卷入太苍旧案、九墟封印、天梯重启与天泉真相，最终重写三界秩序。
- 核心主题：
  - 当资源有限、众生有欲时，秩序应由谁来决定代价。
  - 强者的“正确”是否可以替代众生自己的选择。
  - 真正的休战不是压制一切声音，而是建立可共同承担的规则。

## 已吸收的原始设定
- 境界体系：初照、曲径、幽径、太苍、古道（每升一境增加100年寿命上限），半仙。
- 宗门基础：太苍派为人界正道支柱，分太苍山、长老山、后山禁地三层结构。
- 关键人物：玉玲珑、沈渡衡、顾行舟、叶青棠、陈墨、苏长明（已故）。
- 世界底层：天泉供灵、天庭控天、九墟镇魔、剑圣封天。

## 主角成长线
- 前期：凭天赋和赤雷剑崭露头角，学会“守”。
- 中期：从宗门弟子成长为能看见三界代价的人，学会“疑”。
- 后期：从被棋局推动的人，成长为主动重写规则的人，学会“立”。
