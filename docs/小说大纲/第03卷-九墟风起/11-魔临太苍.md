# 第三卷 第11章 前脚落地后脚至，骨卫撞阵裂纹生

- 核心事件：沈惊蛰和陈墨前脚落地隐剑峰，后脚妖魔大军便已追至。密密麻麻的魔族遮天蔽日，如一片黑色的乌云压向太苍山。厉渊率魔军主力从正面逼近，骨卫——那个全身覆盖灰白骨甲的巨型魔将——以蛮力撞击守山大阵的阵纹节点，大阵开始出现裂纹。太苍山守山大阵自动激活，十二长老紧急就位，各峰峰主率弟子进入战备状态。
- 人物推进：沈渡衡亲自赶到隐剑峰查看情况，在看到昏迷的陈墨和几乎油尽灯枯的沈惊蛰后，面色铁青但没有责备，只命人立刻将二人送去救治。秦望长老在查看妖魔大军的规模后低声对沈渡衡说"这不是普通的兽潮，是一整个墟的魔族"。顾行舟、叶青棠、裴惊鸿、柳沉渊、温如晦等陆续赶到隐剑峰前线。
- 伏笔/钩子：沈惊蛰在被抬走前，用最后一丝灵识感知了一下魔军中领头的几名魔将——厉渊的气息冷酷而沉稳，与陈墨不相上下；另有两三名魔将的气息同样强横。而陈墨此刻正昏迷不醒。
