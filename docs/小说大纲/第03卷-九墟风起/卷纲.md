# 第三卷 九墟风起

## 卷大纲
- 卷核心：主角离开太苍山，沿着西方小派与魔界边缘的线索西行，第一次真正接触人界边疆、裂缝、魔气与九墟传说。他逐渐意识到人、魔并非绝对异族，所谓“魔患”背后牵连着天界旧制与灵气分配之谜。卷末一座边城险些被裂缝吞没，主角虽守下一城，却带回更多关于剑圣法令和九大魔王封印的疑问。
- 主要冲突：表面是追查西方小派的偷袭源头，实质却是太苍旧制、魔界封印与天界灵气秩序三条线第一次在主角眼前交汇。
- 成长主线：主角从“太苍弟子视角”成长为会把凡人、宗门、魔界三方都纳入考量的人，价值观开始真正松动。
- 卷末收束：一城保住了，谜团却更大了；主角带着九墟碑文与旧人遗信回山，也带回了对剑圣与太苍秩序的第一次实质性质疑。
