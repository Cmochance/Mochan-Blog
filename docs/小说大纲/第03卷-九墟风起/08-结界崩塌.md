# 第三卷 第08章 魔主既死界亦亡，厉渊率众出墟来

- 核心事件：魔主死亡的瞬间，心之墟的结界开始崩塌。整个空间剧烈震动，四壁的阵纹逐一熄灭，被封印的妖魔从各个角落苏醒。魔将之首厉渊第一个冲出封印，面容苍白如纸、双瞳漆黑无白，冷酷地扫视了陈墨和沈惊蛰一眼后，发出集结号令。成千上万的妖魔从心之墟中涌出，铺天盖地。
- 人物推进：沈惊蛰在极短的时间内做出判断：留下来等于送死，必须立刻带陈墨撤离。他没有犹豫，背起昏迷的大师兄，以赤雷剑御剑飞行，从正在崩塌的结界缝隙中冲了出去。
- 伏笔/钩子：厉渊在沈惊蛰冲出结界的瞬间看了他一眼，那个眼神没有愤怒，只有冰冷的锁定——他记住了这个人。
