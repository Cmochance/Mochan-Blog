# 第三卷 第07章 最后一剑斩残躯，破魔短剑落尘埃

- 核心事件：第三日末，陈墨以最后一剑斩杀魔主。魔主的残躯在剑光中崩散，化为漫天魔气消散，只留下一柄形制古怪的短剑——破魔之剑，原为琴圣护身短剑，被剑圣封入魔主体内。剑身漆黑如墨，剑刃处隐约有魔纹流转，握在手中时能感觉到一股冰冷而沉重的力量。陈墨将破魔之剑收入怀中，随即因伤势过重而单膝跪地。
- 人物推进：沈惊蛰冲上前扶住陈墨，发现大师兄的经脉已经有多处崩裂，灵力几乎耗尽，身上还残留着大量魔主的心魔之力侵蚀的痕迹。陈墨勉强笑了一下，说"东西拿到了，该走了"。
- 伏笔/钩子：破魔之剑与赤雷剑之间存在某种说不清的联系——两柄剑靠近时，各自的灵光都微微颤动了一下。
