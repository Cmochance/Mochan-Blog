# 第三卷 第14章 五峰灵气汇指尖，一剑斩尽遮天云

- 核心事件：玉玲珑捏出剑诀的瞬间，五峰灵气同时暴动。问剑峰、听泉峰、落霞峰、青云峰、隐剑峰——五座主峰的灵脉在同一瞬间被引动，浩瀚的灵气如五条巨龙从山体中涌出，汇聚在她指尖，凝成一道肉眼可见的白色剑光。一剑。只是一剑。那道剑光从太苍山门前升起，划过天穹，穿透魔云，将遮天蔽日的妖魔大军连同领头的数名魔将一并斩灭。剑光消散之后，天空重新露出湛蓝的底色，方圆百里内的魔气被荡涤一空。
- 人物推进：太苍山上下一片死寂。五千弟子、数十长老、四位峰主，所有人都呆立在原地。没有人说话，没有人动弹，因为他们刚才看见的东西，彻底超出了他们对"隐剑峰峰主"这个身份的全部认知。这不是幽径境能做到的事，甚至不是太苍境能做到的事。
- 伏笔/钩子：玉玲珑收手后身形微微一晃，但立刻稳住了。她引的是五峰灵气而非自身法力，但这一剑的代价仍然不小——她脸色苍白了几分，额角有一滴汗珠滑落。
