# 第三卷 第03章 心魔交织灵识乱，紧随剑影向深行

- 核心事件：二人在心之墟中深入前行，幻象越来越真实——沈惊蛰看见了母亲的身影、听见了苏长明的声音、甚至看到了一个"另一个自己"在裂缝前独自持剑的画面。陈墨一剑劈开每一道幻象，带他穿过层层心魔的侵蚀。
- 人物推进：沈惊蛰在幻象中第一次"看见"了母亲的面容——虽然他知道那是假的，但那一瞬间的情感冲击让他几乎失去理智。陈墨在关键时刻用剑柄敲了他的后脑，冷声道"别看，跟我走"。
- 伏笔/钩子：幻象中出现的母亲面容与沈惊蛰记忆中的模糊印象完全吻合，这让他事后反复思考——心之墟的幻象是凭空捏造的，还是从他灵识深处挖出来的真实记忆？
