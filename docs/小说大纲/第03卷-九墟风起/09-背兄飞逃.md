# 第三卷 第09章 御剑背兄千里路，烛阴幻术追不休

- 核心事件：沈惊蛰背着昏迷的陈墨御剑全速飞行，身后是厉渊率领的妖魔大军。他不惜燃烧灵力加速，但魔将烛阴以幻术沿途伏击——制造分身和假象干扰沈惊蛰的判断，数次在飞行中试图打断他的灵气运转。沈惊蛰靠赤雷剑的雷霆之力震破幻术，但每一次应对都在加速灵力消耗。
- 人物推进：这是沈惊蛰修行以来最绝望的一段路。他的灵力在飞速消耗，经脉因过载而开始出现裂痕，背上的陈墨沉重得像一座山，而身后的魔云始终不曾远离。烛阴忽男忽女的声音不断从四面八方传来，试图瓦解他的意志。
- 伏笔/钩子：飞行途中，陈墨在昏迷中喃喃说了一句话，声音很轻，沈惊蛰没有听清，但他感觉到大师兄怀中的破魔之剑微微发热。
