# 第三卷 第05章 纯粹剑道劈幻象，三日鏖战不退寸

- 核心事件：陈墨与魔主的斗法正式开始。魔主以心魔之术为主要攻击手段，将陈墨一生中最深的恐惧、最痛的记忆、最隐秘的执念一一具象化，试图从精神层面击溃他。陈墨则以二十年磨砺的纯粹剑道应对——不闪不避，每一剑都直指幻象的核心，将其劈成碎片。这场战斗持续了整整三日，陈墨寸步不退。
- 人物推进：沈惊蛰在外围目睹了这场超出自身境界的战斗，看到了陈墨剑道的本质——不是技巧，不是灵气，而是一种将所有花哨剥尽之后只剩下"我要赢"的意志。他也看到了陈墨在幻象中被触及的东西：一个女子的背影、一座燃烧的山门、以及一句"你为什么不留下来"。
- 伏笔/钩子：陈墨在第三日的最后一剑中，剑上隐约浮现出一丝不属于他自身修为的力量，像是有什么东西在暗中相助。
