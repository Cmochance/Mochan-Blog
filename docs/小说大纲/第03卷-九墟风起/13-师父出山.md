# 第三卷 第13章 隐峰深处一人来，寒铁面容怒如海

- 核心事件：一道身影从隐剑峰方向掠来，落在太苍山门前。是玉玲珑。她平日里温和如水的面容此刻沉如寒铁，眼中是沈惊蛰从未见过的、近乎纯粹的愤怒。她没有看任何人，只是抬头望向那片遮天的魔云。太苍山上下所有人都停下了动作，因为他们从未见过这位从不离开隐剑峰的峰主出现在山门前。
- 人物推进：沈渡衡、秦望、裴惊鸿等人看到玉玲珑时的反应各不相同——沈渡衡微微后退了一步，像是在给她让出位置；裴惊鸿握剑的手松开了；秦望低下了头。这些细节让沈惊蛰意识到，太苍山的高层对玉玲珑的真实实力或许早有某种程度的了解。
- 伏笔/钩子：玉玲珑站在山门前，素白长裙在魔气翻涌的狂风中纹丝不动，她缓缓抬起右手，捏了一个剑诀。
