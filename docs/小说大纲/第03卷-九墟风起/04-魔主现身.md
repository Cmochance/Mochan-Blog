# 第三卷 第04章 千年残躯犹可惧，奄奄一息仍称雄

- 核心事件：二人抵达心之墟最深处，魔主现身。这是一个被封印千年、灵力几近枯竭的存在，但即便奄奄一息，它散发出的威压仍让沈惊蛰几乎无法站立。魔主的形态已经不再是完整的生物，更像是一团凝聚了千年怨念与魔气的意志体，在结界的最深处缓慢地消耗着最后的生命力。
- 人物推进：陈墨在看到魔主的瞬间，整个人的气息发生了质变——从收敛到释放，从沉稳到锋锐，像是一柄被压了二十年的剑终于出鞘。沈惊蛰第一次真正感受到大师兄的全部实力，那种压迫感甚至让他想起了玉玲珑。
- 伏笔/钩子：魔主在看到赤雷剑时发出了一声低沉的笑，像是认出了什么旧物。
