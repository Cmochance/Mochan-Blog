# 第一卷 雷池问剑

## 卷大纲
- 卷核心：沈惊蛰入太苍山十五年的成长历程。前三年修行日常与师门交互，第三年追问十年前魔界事件真相未果，被玉玲珑命往雷池守池五年。守池期间拔出赤雷剑、经雷毒淬体、悟出自身剑道、击退妖兽。守池期满后下山历练七年，行走人界见识天地广阔，第七年破境入曲径，归山列入天下会武名单。
- 主要冲突：表层冲突是雷池守池与下山历练中的生死考验，深层冲突是沈惊蛰对母亲之死的疑问、对十年前真相的追问、以及赤雷剑雷毒反噬对经脉的威胁。
- 成长主线：从对太苍山一无所知的新弟子，经历守池的孤独与磨砺、下山历练的见闻与实战，成长为有独立剑道、见过人间百态的修士。
- 卷末收束：沈惊蛰入门十五年归山，修为踏入曲径境，携赤雷剑正式走向天下会武的大舞台。
