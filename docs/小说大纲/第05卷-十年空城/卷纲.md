# 第五卷 十年空城

## 卷大纲
- 卷核心：主角被强行送到十年后的未来，看见一个因自己“消失”而偏离轨道的世界：太苍衰微、裂缝扩散、旧友各自背负代价。这里既是剑圣为他准备的答案，也是另一重更深的骗局。主角在未来搜集真相、修补裂缝，并发现自己对剑圣的怀疑并非空穴来风——只是“坏人”未必等于真正的敌人。
- 主要冲突：外部冲突是灾后世界与裂缝大潮，内部冲突则是主角要不要接受剑圣用强迫换来的“看见真相”，以及他是否还能相信原来的世界观。
- 成长主线：主角从“想改变命运的人”，成长为真正理解改变一个时代需要承担什么代价的人。
- 卷末收束：他带着未来十年的惨烈真相返回现在，决定不再任由任何强者替自己和众生做选择。
