# 第二卷 天下惊雷

## 卷大纲
- 卷核心：四大门派会武开启，主角先在新人乱斗中大放异彩，又因赤雷剑破格打入正赛，与掌门首徒、师姐、西方小派弟子及各方天才交锋。会武既验证了他的雷剑之路，也让七十年前太苍内乱的旧影、西方势力与赤雷剑的价值一并浮出水面。卷末他虽未夺得最高名次，却成为众矢之的，并接到西行查案的宗门任务。
- 主要冲突：明面上是会武争锋，暗线上却是西方小派借比赛报复、旧势力借赤雷剑探路，以及宗门内部对主角是否该被“提前推上台面”的争论。
- 成长主线：主角在高强度连战中学会以脑驭剑、以规则补差距，也开始正视自己与真正老牌天才之间的底蕴鸿沟。
- 卷末收束：会武落幕后，袭杀与查案几乎无缝衔接，主角第一次从宗门新星变成真正意义上的出山行走。
