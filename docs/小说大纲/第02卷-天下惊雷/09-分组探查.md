# 第二卷 第09章 两组深入裂缝地，四人同行各怀思

- 核心事件：秦望将前八名分为两组深入裂缝活跃区探查。沈惊蛰与纪云、乔峥、凌天（萧承安）分在一组，另一组由顾行舟、叶青棠、韩素等人组成。四人在裂缝外围清剿低阶妖兽，配合逐渐默契——纪云以水系术法控场，乔峥正面硬抗，沈惊蛰以雷道收割，凌天则在关键时刻以精准的术法补位。
- 人物推进：沈惊蛰发现凌天的术法路数与任何已知门派都不同，精巧而内敛，像是被刻意压制了锋芒。乔峥对凌天的评价是"这小子不简单，但不像坏人"。纪云则始终保持沉默观察。
- 伏笔/钩子：四人深入到裂缝活跃区核心地带时，脚下的地面开始出现细微的震动，频率越来越密。
