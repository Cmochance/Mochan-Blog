# 第二卷 第11章 暗河辗转灵气异，越深越净非寻常

- 核心事件：四人在地底暗河中辗转求生，沿途发现地下洞穴的灵气浓度远超地面，且越往深处越纯净，与地面上浑浊的魔气渗透截然不同。纪云的水系术法在这种环境中如鱼得水，成为四人前进的关键。沈惊蛰的灵识在纯净灵气中异常敏锐，能提前感知前方的岔路与危险。
- 人物推进：四人在求生过程中建立起初步的信任。乔峥开始称呼沈惊蛰为"雷子"，纪云第一次主动开口说了超过十个字的话。凌天在休息时望着洞壁上的古老纹路出神，像是在辨认什么。
- 伏笔/钩子：洞壁上的纹路越来越密集，风格与任何已知门派的灵纹都不同，沈惊蛰隐约觉得它们与隐剑峰镇魔柱上的阵纹有某种相似之处。
