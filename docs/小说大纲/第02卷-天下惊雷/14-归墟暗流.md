# 第二卷 第14章 韩素言间藏立场，归墟内部意不同

- 核心事件：试炼接近尾声，两组汇合。韩素在与沈惊蛰的交谈中透露归墟宗内部对裂缝问题的立场并不统一——有人主张全力封堵，有人认为应该研究裂缝中外泄的魔气以开发新术法。韩素本人的立场暧昧不明，但在关键时刻从未做过损害同行者的事。
- 人物推进：沈惊蛰第一次意识到，四大门派对裂缝问题的态度远非铁板一块，修仙界的格局比他在太苍山看到的要复杂得多。叶青棠在试炼中与顾行舟有过几次配合，二人的默契让旁人侧目。
- 伏笔/钩子：凌天在试炼结束前悄悄离队了半个时辰，回来后若无其事，但沈惊蛰注意到他袖口沾了一点不属于这片区域的泥土。
