# 第二卷 第13章 三丈之内不可近，无形之力送归途

- 核心事件：沈惊蛰试图靠近石台，但距离长枪三丈时便被一股无形之力推回，灵识也被完全屏蔽。乔峥尝试以体修之躯硬闯，被弹飞得最远。四人在空间中停留不到半柱香，地面忽然亮起一道阵纹，一股温和却不可抗拒的力量将他们托起，穿过层层岩壁，直接送回地面。
- 人物推进：回到地面后四人发现身处裂缝活跃区数十里外的荒野中，身上毫发无伤，连衣物上的尘土都被清理干净。入口已消失无踪。乔峥笑称"那枪看不上我这种粗人"，纪云则反复追问沈惊蛰在靠近时感受到了什么。
- 伏笔/钩子：沈惊蛰回想起被推回的瞬间，那股力量并非敌意，更像是一种"不是时候"的拒绝——与玉玲珑当年拒绝回答他问题时的态度如出一辙。
