# 第四卷 太苍旧梦

## 卷大纲
- 卷核心：回山之后，太苍表面的安稳被后山禁地、长老山旧案和掌门寿关危机一一撕开。主角围绕赤雷剑、剑圣分身和七十年前内乱真相不断深入，发现宗门内部有人仍想借裂缝和天梯改写世道。卷末剑圣逼他踏入一场超越时间的棋局，让“主角为何会消失十年”的谜题正式登场。
- 主要冲突：核心冲突从外部敌人转向宗门内部：太苍到底是在守护世界，还是在用牺牲少数人的方式强压危机。
- 成长主线：主角开始学会与“制度”“历史”“大人物的理由”对抗，他的反抗不再是少年意气，而是建立在亲眼所见之上的判断。
- 卷末收束：太苍夜战之后，剑圣本人出手镇压一切，并以近乎强迫的方式把主角送去十年后的未来。
