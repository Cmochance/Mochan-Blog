# 人物设定

## 境界体系

凡人寿命约百年，踏入修行后每提升一层境界可增加一百年寿命上限。

| 境界 | 名称 | 寿命上限 | 说明 |
|------|------|----------|------|
| 第一境 | 初照 | 200年 | 灵气初入经脉，感知天地，踏入修行之门 |
| 第二境 | 曲径 | 300年 | 灵气贯通曲折经脉，可驭术法，初具战力 |
| 第三境 | 幽径 | 400年 | 灵气深入幽微之处，术法精深，可称强者 |
| 第四境 | 太苍 | 500年 | 灵气与天地共鸣，修为通达，宗师之境 |
| 第五境 | 古道 | 600年 | 触及上古大修士之道，已近仙人门槛 |
| — | 半仙 | 近乎无限 | 超越五境，寿元几近无穷，世间罕有 |

每一境内部分为十境（一境至十境），突破十境即可进入下一大境界。

## 七圣

### 琴圣与九世轮回

琴圣在封印九墟时感染魔毒，毒入灵脉，灵气不散尽则魔毒不消。剑圣为救琴圣，设计了九世轮回之法：将琴圣残存灵气凝聚、封印记忆、重塑身躯，令其每一世以全新身份修炼。每一世通过不断吸收纯净灵气、消耗被魔毒感染的浑浊灵气，逐渐纯净化灵脉。

### 枪圣

- 身份：七圣之一，存世
- 长相：待定
- 性格：待定
- 境界成长：两千年前登仙又返回人间
- 设定：在地下深处创造灵界，维持天地灵气循环。灵界入口以枪圣之枪为封印，非灵体无法进入。

### 剑圣

- 身份：七圣之一，存世，太苍山真正的守护者
- 长相：待定
- 性格：沉默寡言，行事果决到近乎冷酷，信奉"以最短的刀劈开最长的乱世"
- 境界成长：两千年前登仙又返回人间，修行千年世间无敌
- 设定：隐居太苍山后山，以无上大能封闭天梯、镇压魔界、制定太苍境法令。为救琴圣设计九世轮回，为加快进度捏造无记忆分身苏长明。

### 书圣

- 身份：七圣之一，存世
- 长相：待定
- 性格：待定
- 境界成长：两千年前登仙又返回人间
- 设定：进入凡人唯一王朝担任萧承安老师，单独传授仙法和知识，守卫王朝稳定，维持凡间秩序。

### 弓圣

### 棋圣

- 设定：棋隐山为其遗脉。其余待定。

- 身份：七圣之一，已陨落

## 太苍派

### 隐剑峰

#### 玉玲珑

- 身份：隐剑峰峰主，琴圣第八世，沈惊蛰师父
- 长相：面容年轻如二十余岁女子，一头黑发（入道时曾一夜白头，破古道后由白转黑），气质温和却不可亲近
- 性格：表面温柔平和，实则意志坚如磐石，对弟子关爱但从不溺爱，涉及过去之事时沉默而不可动摇
- 境界成长：20岁入门→一年入道一夜白头→三年入太苍境→五年破古道转黑发成半仙。天赋之高实为琴圣八世轮回灵脉趋近纯净的体现
- 设定：至今修行已千年（累计九世），世间除剑圣外无人能敌。10年前因抓住苏长明的手导致一魂一魄被带入镇魔柱，只能在隐剑峰一定范围内动用全部法力。招收弟子由她独自决定，隐剑峰无其他长老。

#### 苏长明（剑仙）

- 身份：剑圣无记忆分身，玉玲珑师父，已故
- 长相：面如冠玉，气质清冷出尘，常着白衣
- 性格：天性洒脱，行事全凭本心，对玉玲珑既是师父又如兄长，温和中带着不自知的孤高
- 境界成长：60多年前入门→极短时间内成为人间第一，被世人称为"剑仙"
- 设定：剑圣为加快琴圣轮回纯净化进度而捏造的分身，本人并不知晓自己的真实身份。10年前带玉玲珑前往魔界镇压心之魔域，发现沈惊蛰并将其带离；回山后牺牲自己加固镇魔柱，肉身消散融入柱中。

#### 陈墨（大师兄）

- 身份：隐剑峰大师兄，玉玲珑首徒
- 长相：身形高大，面容棱角分明，常年带着一丝懒散的笑意
- 性格：沉稳中偶有玩世不恭，对师弟妹照顾有加，对师父既敬又畏，实力深藏不露
- 境界成长：20年前以幽径境高阶修为被玉玲珑击败，依赌约拜师→现为幽径十境巅峰，差一步破境入太苍
- 设定：入门前已是人界顶尖强者，拜师后二十年修为更进一步。现为太苍派第一战力，人间除隐世高手外少有敌手。对太苍山内外局势了解最深，是隐剑峰中除玉玲珑外唯一能独当一面的存在。

#### 苏念卿（二师姐）

- 身份：隐剑峰二师姐，陈墨入门后第一个收的师妹
- 长相：容貌清丽，眉目间有几分英气，束发利落，常佩一柄窄刃短剑
- 性格：冷静自持，话不多但句句在理，对师弟妹严格却公正。外人面前是隐剑峰的门面担当，私下偶尔会露出毒舌的一面
- 境界成长：15年前入门时为初照境→现为曲径八境，修行速度在同辈中属上乘
- 设定：擅长近身剑术与灵识探查，是隐剑峰中最适合执行侦查任务的人。与陈墨配合默契，二人常被派出处理宗门外务。

#### 卫朝歌（三师弟）

- 身份：隐剑峰三师弟
- 长相：面容清秀，身形偏瘦，看着比实际年龄小，手指常沾着墨渍或灵纹粉
- 性格：好奇心重，爱钻研灵纹与阵法，话多但不讨人厌，是隐剑峰的气氛调节者。遇到感兴趣的阵纹会忘记吃饭睡觉
- 境界成长：12年前入门时为初照境→现为曲径三境，修行进度偏慢，但灵纹造诣远超同境修士
- 设定：师从玉玲珑却更痴迷阵法而非剑道，常去青云峰旁听温如晦讲阵理。是隐剑峰中唯一能维护镇魔柱外围阵纹的人，对镇魔柱的结构比其他师兄弟了解更多，但被玉玲珑严令禁止深入研究。

#### 姜时雨（四师妹）

- 身份：隐剑峰四师妹
- 长相：圆脸杏眼，笑起来有酒窝，身量不高，常穿浅色衣裙
- 性格：性子软但韧性足，不争不抢却有自己的坚持。是隐剑峰中最能安抚人心的存在，师兄弟受伤或心情低落时第一个想到的人
- 境界成长：11年前入门时为初照境→现为曲径二境，修行天赋平平但胜在稳扎稳打
- 设定：擅长水系辅助术法与疗愈之术，曾在听泉峰柳沉渊处学过丹道基础。是隐剑峰中唯一能配制解毒丹药的人，沈惊蛰守雷池期间雷毒发作时，多次为他远程配药。与卫朝歌关系最好，常被对方拉去当阵法实验的"观众"。

#### 沈惊蛰（五师弟 / 主角）

- 身份：隐剑峰五师弟，玉玲珑弟子
- 长相：面容普通偏清瘦，眼神沉静，不笑时显得冷淡，笑起来却很温和
- 性格：内敛寡言但内心炽热，对母亲的死和师父的沉默有深深的疑问，重情重义，遇事先救人后问缘由
- 境界成长：10年前入门→前三年初照境→守雷池五年拔出赤雷剑→下山历练七年→第十五年破境入曲径
- 设定：神族与魔族结合的后裔，身上兼有修仙者灵气与微弱魔气。10年前苏长明在魔界发现他时，因其身上灵气远强于魔气，判断他不是魔族而将其带出。在人界修行多年后每日吸纳灵气，体内魔气已微乎其微，寻常修士根本无法辨别。世间只有现存三圣和第九世恢复的琴圣能以灵识直接解析出他的真实血脉。玉玲珑并不知道他的真实身份，只知道他的母亲死于十年前的魔界事件。对灵气运用有独特敏感（源于神族血脉），悟出自身剑道——迅烈、霸绝，出剑附带雷霆麻痹与破灵护体之效。

### 问剑峰
